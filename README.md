# 🐄 Qurbani Hat Platform

## 📌 Project Name
Qurbani Hat Ashy

## 🌐 Live URL
https://qurbani-hat-ashy.vercel.app/

## 🎯 Purpose
The Qurbani Hat Platform is an online marketplace designed to facilitate the buying and selling of sacrificial animals (Qurbani) during Eid-ul-Adha. It provides users with a simple and user-friendly interface to browse animals, view details, and connect with sellers.

## 🚀 Key Features
- 🐐 Browse available Qurbani animals
- 📋 Detailed information for each animal (price, category, description)
- 🔍 Easy navigation and clean UI
- 📱 Responsive design for mobile and desktop
- ⚡ Fast performance with modern frontend technologies
- 🛒 Basic e-commerce style layout for better user experience

## 🛠️ Technologies Used
- React.js
- Next.js
- Tailwind CSS / CSS Framework
- JavaScript (ES6+)
- Vercel (Deployment)
